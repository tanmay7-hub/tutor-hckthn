# TutorIQ — Teal + Galaxy Theme

- Order: Home → Profile → Motive → Dashboard → Chatbot → Quiz
- Landing renamed to **Home**, Onboarding renamed to **Motive**
- Global font: **Nunito Sans** (ExtraBold used for headings)
- Theme: Teal accents + Galaxy gradient
- Profile fields: Name, Age, Gender, Studying status (saved in localStorage)
- Dashboard: Glowing **Edit Roadmap** button
- Buttons: Modern, glowy hover
- Chatbot UI: Bubbles, sleek header
- Fully responsive with Tailwind
- Smooth page transitions with Framer Motion

## Run
```bash
npm i
npm run dev
```
